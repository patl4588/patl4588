# Pat — building MeterCall

> **MeterCall** — a pay-per-call module marketplace. 1,000+ modules replacing $20K–$36K/yr SaaS incumbents at ~99% less.

### Live
- [metercall.ai](https://metercall.ai) — homepage + module browser
- [metercall.ai/alternatives/](https://metercall.ai/alternatives/) — 50 head-to-head comparisons

### Why
Most SaaS seats run at 5–15% utilization. You pay for 100%. MeterCall flips that — you pay only for the calls you make.
